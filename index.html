# 快速开始

环境准备

请确保你的 Node.js 版本 >= 8.6，并已经安装Yarn工具

​ 　项目采用VuePress框架编写在线文档，首先，需要执行如下命令完成项目安装并启动。

# 初始化项目文档
startup.bat -init
# 启动命令
yarn dev

​ 　启动完成后，即可通过http://localhost:8001/funtl/访问本项目。若想终止运行，请输入ctrl + c 。

​ 　最后，您可以参考这里对文档的默认主题（包括导航栏、侧边栏）进行修改。修改完成后，可以通过如下命令，将修改结果部署到GitHub Pages上，部署原理请参照这里。

# 推送文件到GitHub Pages中
startup.bat -push
